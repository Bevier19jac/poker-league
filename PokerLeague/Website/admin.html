How To Deploy (GitHub Pages)

One-time setup (5 min):
  1. Go to github.com → Sign up free (or log in)
  2. Click + → New repository → name it poker-league → Public → Create
  3. Click Add file → Upload files → drag all 4 files from your PokerLeague/Website/ folder
  4. Click Commit changes
  5. Go to Settings → Pages → Source: Deploy from branch → Branch: main → /root → Save
Your site is live at: https://yourusername.github.io/poker-league

Update After Each Game

  1. Open this admin page → Post Game Results tab → fill in results → Generate Code → Copy
  2. Open github.com → your repo → click data.js
  3. Click the pencil icon (Edit this file)
  4. Find lastGame: { and replace that whole block with your copied code
  5. Scroll down → click Commit changes → site updates in ~30 seconds

Add Player Avatars

  1. Take a screenshot from each player's interview video (face forward, good lighting)
  2. Crop to square, save as firstname.jpg (lowercase) — e.g. tod.jpg, chrisfr.jpg
  3. In data.js, find that player in standings and add: avatar: "assets/avatars/tod.jpg"
  4. Upload the photos to GitHub: repo → Add file → Upload → drop the jpg files into assets/avatars/
